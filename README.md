# Template Portfolio built with React 

This is a preset developer portfolio for any developer who is new in programming and don't want to rush at designing side.


## Live demo

https://template-portfolio-qoevc7kun-mahmedsid.vercel.app/


## Tech Stack

**Client:** React, Chakra UI, Formik, Yup




## Improvements

There is alot of Improvements sides in that you can explore the dim points and craft it with your desired theme. Chakra UI can help to make it superb as it is pre-installed.


## Deployment

This Project is Deployed on Vercel.

To run this project on you local system, follow these steps

1. run command in terminal.
```bash
  npm install
```
2. Then run 
```bash
npm start
```
and server will easily run.
